# Employee Management System (EMS)

## Description

This is a simple Python project to manage employee data.
You can add, view, and search employees using this program.

---

## Features

* Add employee
* View all employees
* Search employee by ID
* Exit program

---

## How to Run

1. Make sure Python is installed
2. Run the file:

```
python main.py
```

---

## Example Menu

```
1. Add Employee
2. View All Employees
3. Search Employee
4. Exit
```
